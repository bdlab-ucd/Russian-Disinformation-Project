This directory contains CSV data files that were obtained from the Internet Research
Agency Ads project at the University of Maryland.

    https://mith.umd.edu/irads/data.zip"j

The dataset includes two files:

- items.csv: a table of each Facebook advertisement and its properties
- tag-matrix.csv: a matrix of tag combinations from the codebook

In addition the data is available through the a REST API endpoint:

    https://mith.umd.edu/irads/api/

For more information about the API please consult the Omeka documentation:

    https://omeka.readthedocs.io/en/latest/Reference/api/index.html
